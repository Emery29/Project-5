📘 What Are Linear Relationships?

Linear equations and linear functions describe relationships that form straight lines when graphed. They appear everywhere — finance, physics, business, and everyday problem‑solving.

✏️ Linear Equations

A linear equation is an equation whose graph is a straight line. It has the form:

ax + b = c

Example

Solve: 2x + 6 = 14

Steps:

  • Subtract 6 from both sides → 2x = 8
  • Divide by 2 → x = 4

📈 Linear Functions

A linear function describes how one quantity changes with another. It has the form:

f(x) = mx + b
  • m = slope (rate of change)
  • b = y‑intercept (starting value)
